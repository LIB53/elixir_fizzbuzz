# ElixirFizzbuzz

Elixir implementation of Jeff Atwood's ["FizzBuzz" test](https://blog.codinghorror.com/why-cant-programmers-program/).

## Compile & Run
1. Compile and open iex shell
```shell
iex -S mix
```

2. In iex shell, run Fizzbuzz.start
```shell
iex(1)> Fizzbuzz.start
```

## Tests
```shell
mix test
```
